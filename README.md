# Trang Web Tình Yêu

Đây là trang web đơn giản để gửi tin nhắn tình yêu qua email.

## Cách sử dụng

1. Truy cập trang web
2. Nhập email người nhận
3. Nhấn nút "Gửi Tin Nhắn"
4. Kiểm tra hộp thư email để nhận tin nhắn

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript
- EmailJS API
